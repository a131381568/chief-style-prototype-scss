# Chief 原始 SCSS 設定
此為 chief 開發靜態 HTML 時的 SCSS 設定

## main
每個頁面的 scss 檔設定來源
- 變數宣告
- RWD 設定
- 全域基礎設定
- 元素樣式
- 網站頁首
- 首頁 ( Header )
- 單一頁面統一設定 ( Single Setting )
- 分類彙整頁 ( Category Archive )
- 單一文章 ( Single Post )
- 機房小首頁
- 解決方案小首頁
- 單一解決方案
- 影片小首頁
- 頁尾 ( Footer )

## variable
統一的變數宣告、繼承樣式、客製化函式

## basic
`body` 下的全域設定

## media
- 統一管理 RWD 所需的螢幕寬度區間範圍
- 條列並客製化每個區間範圍的函式
